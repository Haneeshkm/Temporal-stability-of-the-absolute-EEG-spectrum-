# Temporal Stability of the Absolute EEG Spectrum During Mindfulness Meditation

This repository contains the Python analysis pipeline used to investigate temporal EEG spectral dynamics during a 30-minute mindfulness meditation session in experienced Buddhist monks.

## Main Analyses

- EEG spectral power extraction
- ROI-based analysis
- Linear mixed-effects modelling
- Relative power analysis
- Compositional data analysis (CLR)
- Alpha trajectory clustering
- Random Forest classification
- Statistical validation

## Dataset

The study uses the publicly available EEG meditation dataset from:

Wongupparaj P. (2024)
https://doi.org/10.17632/pthdhf2dwm.1
